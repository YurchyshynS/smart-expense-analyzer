# Smart Expense Analyzer

## Description
Smart Expense Analyzer is a Python desktop AI system that analyzes personal expenses, predicts spending trends, and evaluates budget risk using machine learning techniques.

## Features
- Expense analysis (Food, Transport, Shopping)
- Budget risk classification
- Linear regression forecasting
- Graph visualization
- Tkinter GUI

## Technologies
- Python 3.10+
- Tkinter
- NumPy
- Matplotlib

## How to run
```bash
pip install -r requirements.txt
python main.py
